**Problem:** Pick one number from `arr1` and one from `arr2`. Return the length of the **longest common numeric prefix** (digits from the left).

Example: `123` and `129` share prefix `12` → length **2**.  
Example: `1` and `1000` share prefix `1` → length **1**.

---

## Approach

1. **Store every prefix** of every number in `arr1` in a `set`.
2. **Scan `arr2`:** for each number, generate its prefixes the same way; if a prefix is in the set, update the answer with that prefix’s **digit count**.

A number’s prefixes are obtained by stripping digits from the right with `num // div`, where `div` starts at `10^(digits-1)` and `div //= 10` each step.

Digit count: `floor(log10(num)) + 1` (valid here since all values are `≥ 1`).

---

## Code

Checkout the two loops below — first builds the set, second checks `arr2` and tracks max prefix length.

```python title:longestCommonPrefix.py
import math

class Solution:
    def longestCommonPrefix(self, arr1: List[int], arr2: List[int]) -> int:
        prefixes = set()

        for _, num in enumerate(arr1):
            digits = math.floor(math.log10(num)) + 1
            div = 10 ** (digits - 1)
            while div > 0:
                prefixes.add(num // div)
                div = div // 10

        ans = 0
        for _, num in enumerate(arr2):
            digits = math.floor(math.log10(num)) + 1
            div = 10 ** (digits - 1)
            while div > 0:
                curr_px = num // div
                if curr_px in prefixes:
                    ans = max(ans, math.floor(math.log10(curr_px)) + 1)
                div = div // 10
        return ans
```

**TC:** `O((n + m) · d)` — `d` ≤ 10 for `num ≤ 10^9`  
**SC:** `O(n · d)` for the set
